
# Project Title

Get daily news-a news application

## Installation

create react app using npm create vite@latest newsapp

```bash
  npm install 
  cd newsapp
```
## Dependencies
axios: 1.6.7,
react: 18.2.0,
react-dom: 18.2.0,
react-router-dom: 6.21.3,
vite-plugin-env-compatible: 2.0.1
